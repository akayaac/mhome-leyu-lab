# mhome-leyu-lab

本仓库用于归档和发布多个独立的 HTML 页面。这些页面可能包含工具、演示、实验性项目或静态内容，不针对任何特定域名或网站。

## 项目简介

mhome-leyu-lab 是一个个人项目归档仓库，主要存放以 HTML 格式为主的静态页面。每个页面通常独立存在，不依赖后端服务或数据库。适合用于存放前端实验、小工具原型、文档型页面等。

## 目录说明

```
├── pages/           # 存放各独立 HTML 页面及关联资源
├── assets/          # 通用资源文件（如 CSS、JS、图片等）
├── index.html       # 仓库首页或导航页（可选）
└── README.md        # 本文件
```

- `pages/` 目录下每个子目录或文件对应一个独立页面。
- `assets/` 目录存放多个页面可能共用的资源。
- 若存在 `index.html`，可作为浏览入口或页面列表。

## 页面归档说明

每个独立页面通常包含一个完整的 HTML 文件及其所需的样式、脚本和资源。页面之间互不关联，可单独访问。归档时建议：

- 每个页面放在独立的子目录中，便于管理关联资源。
- 页面文件名保持简洁，避免特殊字符。
- 如需说明页面用途，可在页面内注释或另附说明文件。

## 维护说明

- 本仓库不定期更新，新增或修改页面。
- 不保证所有页面长期可用或兼容所有浏览器。
- 欢迎 fork 或参考代码，但不对页面内容作任何形式的担保。
- 如有问题或建议，可通过 Issues 提出。
